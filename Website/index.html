Agent Roles

Each agent masters one thing

Specialization beats generalization. Every agent in Datrix has a single, well-defined responsibility.

Agent 01

Query Agent

Translates your plain-English request into a structured collection specification. Determines scope, fields, sources, and volume before a single byte is scraped.

Agent 02

Collector Agents

Multiple parallel agents — each assigned a specific source. Websites, directories, maps, public databases. They run simultaneously, never wait for each other.

Agent 03

Verifier Agent

Receives all collected data and cross-checks every field across sources. Detects duplicates, resolves conflicts, and assigns a per-field confidence score.

Agent 04

Human-in-the-Loop

Low-confidence entries never silently slip through. They surface in a clean review interface — you confirm, correct, or skip. The system learns from every decision.

Agent 05

Output Agent

Formats your verified dataset into exactly the structure you need — CSV, Excel, or JSON — with a confidence column on every single row. No surprises in production.
